Cadastrar Novo Funcionário

Acesso completo ao dashboard interno

1

Acesse o sistema como Admin

Acesse dalacortefs.com.br/login com sua conta de administrador.

2

Via API — endpoint de criação de usuário

Atualmente o cadastro de usuários é feito via API diretamente. Use o seguinte endpoint:

POST https://dalacortefs.com.br/api/auth/register

3

Dados necessários (JSON)

{
  "name": "Nome Completo do Funcionário",
  "email": "funcionario@dalacortefs.com.br",
  "password": "Senha@2024",
  "password_confirmation": "Senha@2024",
  "tipo": "funcionario"
}
4

Teste com Postman, Insomnia ou curl

Exemplo com curl:

curl -X POST https://dalacortefs.com.br/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"name":"João Silva","email":"joao@dalacortefs.com.br","password":"Senha@2024","password_confirmation":"Senha@2024","tipo":"funcionario"}'
📌 Recomendação de melhoria Para facilitar o cadastro sem precisar de API, podemos criar uma tela de gestão de usuários no dashboard (menu Configurações → Usuários). Me peça para implementar quando quiser.

Cadastrar Novo Cliente

Acesso ao Portal do Cliente

1

Cadastro de usuário tipo "cliente"

Mesma rota de registro, com "tipo": "cliente":

{
  "name": "Nome do Cliente",
  "email": "cliente@empresa.com.br",
  "password": "SenhaSegura@123",
  "password_confirmation": "SenhaSegura@123",
  "tipo": "cliente"
}
2

Cadastrar o Cliente na tabela de Clientes

Além do usuário, cadastre o cliente no módulo de Clientes do dashboard (menu Clientes → Novo Cliente) com os dados completos: CNPJ/CPF, razão social, telefone, endereço, regime tributário.

3

Enviar credenciais ao cliente

Após cadastrar, envie para o cliente:

🔗 Link de acesso: dalacortefs.com.br/login

📧 E-mail: o e-mail cadastrado

🔑 Senha: a senha definida no cadastro

O cliente será direcionado automaticamente para o Portal do Cliente ao fazer login.

4

O que o cliente verá no Portal

AbaConteúdo
ResumoTotal de entradas, saídas e saldo líquido dos extratos cadastrados
ExtratosLista de extratos bancários enviados pelo escritório
NotíciasPublicações do módulo de Notícias do dashboard
Assistente IAChat com IA para dúvidas financeiras e contábeis

Como Testar o Sistema Agora

Passo a passo para validar se tudo está funcionando

1

Testar o login de Admin

Acesse dalacortefs.com.br/login
Email: admin@dalacortefs.com.br
Senha: Admin@2024
Resultado esperado: redireciona para o dashboard interno.

2

Verificar se o backend está respondendo

Acesse no navegador: https://dalacortefs.com.br/api/health ou /api/auth/me
Se retornar JSON, o Laravel está funcionando.
Se retornar erro 500 ou página em branco, o backend ainda precisa de configuração no servidor.

3

Testar o login de Cliente

Email: cliente@dalacortefs.com.br
Senha: Cliente@2024
Resultado esperado: redireciona para dalacortefs.com.br/portal

4

Se o login falhar (erro de credenciais)

O banco de dados pode não ter sido populado com os dados de seed. No servidor (HostGator via SSH ou terminal cPanel), execute:

cd ~/laravel
php artisan migrate --force
php artisan db:seed --force
🔧 Backend ainda não está configurado? Se o login não funcionar, é porque o Laravel (backend) ainda precisa ser instalado no servidor separadamente. O deploy atual só subiu o frontend (site estático). Me peça para configurar o backend no servidor quando estiver pronto para isso.
